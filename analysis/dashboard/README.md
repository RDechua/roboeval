# RoboEval — Phase 5 Dashboard

Interactive dashboard for the RoboEval failure-mode and residual-RL study
of ACT on AlohaTransferCube. See the main repo at
[github.com/RDechua/roboeval](https://github.com/RDechua/roboeval)
for the full PRD, research log, and source.

**First visit:** if the Space has been idle for more than 48 hours, the
container needs about 30 seconds to wake. Subsequent loads finish in
under 3 seconds (PRD §9.1 acceptance).
